# Open Source Asset Leads

Prefer these sources when replacing placeholder visuals in production.

- Kenney - Isometric Miniature Farm. CC0. Good fit for buildings, plots, trees, and decorative props. https://kenney.nl/assets/isometric-miniature-farm
- Kenney - Pixel Platformer Farm Expansion. CC0. Useful for crops, greenhouse parts, and farm-themed pixel tiles. https://kenney.nl/assets/pixel-platformer-farm-expansion
- OpenGameArt - Pixel farm and shack. CC0. Small isometric farm pack for quick mockups. https://opengameart.org/content/pixel-farm-and-shack
- OpenGameArt - Farming Set (Pixel-Art). CC0. Includes produce, animals, and simple farming elements. https://opengameart.org/content/farming-set-pixel-art
- OpenGameArt - [LPC] Farm. CC-BY 4.0. Larger coherent world pieces such as barns, silos, sheds, and machines. https://opengameart.org/content/lpc-farm
- OpenGameArt - [LPC] Crops. CC-BY-SA 4.0 / CC-BY-SA 3.0 / GPL 3.0, with some assets noted separately. Strong source for crop growth stages. https://opengameart.org/content/lpc-crops

Notes:
- QQ Farm and Stardew Valley are proprietary visual references, so use them as style inspiration only.
- Prefer CC0 packs first if you want the least licensing friction for deployment and agent reuse.
- If you mix CC-BY or CC-BY-SA packs, keep a single credits file and confirm share-alike requirements before production use.
